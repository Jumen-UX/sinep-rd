# Manuales de SINEP RD

> Estado: vigente para beta interna
> Última revisión: 2026-07-20

Esta carpeta contiene la documentación operativa del sistema.

## Documentos

- [Manual de usuario](./manual-de-usuario.md): consulta pública, navegación, creación y edición autorizada, revisión e importaciones.
- [Manual de administrador](./manual-de-administrador.md): usuarios, permisos, estructuras, cargos, auditoría, calidad de datos y operación administrativa.

## Estado documental

Los manuales son documentos vivos y deben actualizarse cuando cambie cualquiera de estos elementos:

- Flujo funcional.
- Pantalla o navegación.
- Rol, permiso o alcance.
- Regla de negocio.
- Modelo de datos visible para el usuario.
- Mensaje de error o procedimiento de recuperación.

## Criterio para publicación estable

Antes de cada versión pública:

1. Validar los procedimientos contra la aplicación desplegada.
2. Incorporar capturas de pantalla actualizadas.
3. Confirmar nombres exactos de menús, botones y estados.
4. Revisar accesibilidad del contenido.
5. Registrar la versión de la aplicación y la fecha de revisión.
6. Exportar una edición distribuible en PDF cuando corresponda.
